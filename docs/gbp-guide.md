# Google Business Profile — Guide Pizza Deli'Zza

## 1. Création du profil

1. Rendez-vous sur [business.google.com](https://business.google.com)
2. Connectez-vous avec un compte Google dédié à l'entreprise
3. Cliquez **Ajouter votre établissement** → **Ajouter un seul établissement**
4. Saisissez **Pizza Deli'Zza** comme nom d'établissement
5. Choisissez la catégorie principale : **Pizzeria** (voir section catégories)
6. Indiquez l'adresse : **98 Place du Jardin Public, 49270 Orée d'Anjou**
7. Choisissez **Non** pour « Livrez-vous à l'adresse de vos clients ? »
8. Ajoutez le numéro de téléphone et le site web `https://delizza.fr`
9. Validez l'établissement (carte postale, téléphone ou e-mail selon proposition Google)

## 9. Stratégie avis Google

### Objectif
Atteindre **50 avis 5 étoiles** dans les 3 premiers mois.

### QR code
1. Générez un lien court vers votre fiche GBP (Google fournit un lien « Demander des avis »)
2. Créez un QR code avec ce lien (via qr-code-generator.com ou similaire)
3. Imprimez-le sur :
   - Un petit chevalet posé sur le comptoir
   - Les emballages pizza (sticker ou tampon)
   - Un flyer A6 glissé dans chaque commande

### Script demande d'avis

**À l'oral (au comptoir) :**

> « Merci pour votre commande ! Si vous avez apprécié nos pizzas, un petit avis Google nous aiderait énormément. Vous pouvez scanner le QR code sur votre emballage, ça prend 30 secondes ! »

**Par message (si collecte d'emails/SMS) :**

> Bonjour ! Merci d'avoir choisi Pizza Deli'Zza 🍕
>
> Votre avis compte beaucoup pour nous. Si vous avez aimé nos pizzas, pourriez-vous nous laisser un avis Google ? Cela nous aide à faire connaître notre pizzeria artisanale.
>
> 👉 [Lien vers avis Google]
>
> Merci et à bientôt !
> L'équipe Pizza Deli'Zza

### Bonnes pratiques
- Répondre à **tous** les avis (positifs et négatifs) dans les 24h
- Réponses personnalisées, jamais de copier-coller identique
- Ne jamais proposer de contrepartie en échange d'un avis
- Signaler les faux avis via l'interface GBP

4. Une fois vérifié, soumettez le sitemap : `https://delizza.fr/sitemap.xml`
5. Surveillez les erreurs d'indexation dans l'onglet **Pages**
